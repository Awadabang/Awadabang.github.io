Golang Generate CRUD code from SQL

发表于2023-03-02|更新于2022-03-03
|阅读量:

1.CRUD包

  • database/sql
    • 非常快、代码简单
    • 必须手动将SQL字段映射到变量
    • 容易出错，执行时才会发现错误
  • gorm
    • CRUD函数已经实现，生产代码会很短
    • 必须学习gorm提供的函数编写查询
    • 流量大时，运行很慢
  • sqlx
    • 很快、易用
    • 字段映射通过查询文本或结构标签完成的
    • 错误只会在运行时被捕获
  • sqlc
    • 很快、易用
    • CRUD代码自动生成
    • 错误会被发现并立即报告
    • 目前只适用于PostgreSQL和MySQL
